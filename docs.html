engine — Walkthrough 讲解引擎

逐步高亮 + 动画 + 语音旁白 + 键盘控制。用于技术选型报告、原型讲解、架构图等。

引用

<link rel="stylesheet" href="https://boom-cdn.vercel.app/engine/engine.css">
<script src="https://boom-cdn.vercel.app/engine/engine.js"></script>

初始化

SR.init({
  steps: [
    {label: '标题', sub: '副标题', audio: 's1'},
    {label: '第二步', sub: '说明'}
  ],
  animations: [
    async () => { SR.showScreen('s-page1'); await SR.wait(500); },
    async () => { SR.glow(SR.$('elem-id')); await SR.playAudio('s2'); SR.unglow(SR.$('elem-id')); }
  ],
  audioDir: 'audio/demo-name/',
  title: 'Demo Title',
  meta: '10 steps · description',
  device: 'presentation'
});

HTML 结构

<div class="page">
  <div class="sr-screen active" id="s-page1">...</div>
  <div class="sr-screen" id="s-page2">...</div>
</div>

每个 .sr-screen 是一页。引擎通过 SR.showScreen(id) 切换。

API

SR.$(id) — getElementById SR.wait(ms) — 可中断等待 SR.showSub(text) — 显示字幕 SR.hideSub() — 隐藏字幕 SR.playAudio(id) — 播放音频 SR.glow(el) — 蓝色高亮 SR.unglow(el) — 取消高亮 SR.showScreen(id) — 切换页面 SR.highlight(el, ms?) — 框选高亮 SR.center(el) — 返回中心坐标

键盘

Space播放/暂停
← →上一步/下一步
M静音
R阅读模式（全页纵向浏览）
Esc退出阅读模式

viewer — 查看器

keyframe-viewer（关键帧查看器）

用于原型图和流程图的逐帧讲解。左侧画面 + 右侧讲解面板 + 高亮标注。

<link rel="stylesheet" href="https://boom-cdn.vercel.app/viewer/keyframe-viewer.css">
<script src="https://boom-cdn.vercel.app/viewer/keyframe-viewer.js"></script>

前置：定义全局 frames 数组，DOM 中需要 #hlRing #hlLabel .view-panel 等元素。

HTML 骨架参考：keyframe-viewer.html

manifesto-viewer（宪法渲染器）

从 manifesto.json 自动生成项目宪法页面。

# 下载渲染器到项目
curl -o spec/v{X}/final/manifesto/index.html \
  https://boom-cdn.vercel.app/viewer/manifesto-viewer.html

# 同目录放 manifesto.json，打开 index.html 自动渲染

查看模板：manifesto-viewer.html

version-index（版本索引）

版本导航首页，列出原型/流程图/BDD/宪法等产出物。

curl -o spec/v{X}/final/index.html \

查看模板：version-index.html
