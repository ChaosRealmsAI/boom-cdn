What is Boom CDN?
A themeable CSS component library. One link tag, 40+ class-based components, 8 curated themes. No build step, no dependencies.
How do I add a custom theme?
Create a CSS file that imports base.css and overrides the CSS custom properties. All design tokens are controlled by variables.
Is it accessible?
The collapsible uses native HTML <details> / <summary> — keyboard accessible by default in all browsers.
