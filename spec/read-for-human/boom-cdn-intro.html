从零到可用只需要三步

三步上手

01

选一个主题 CSS

18 套主题各有风格，从赛博朋克到瑞士极简。每个主题是一个 CSS 文件，它会自动引入 base.css 组件框架，所以你只需要链接一个文件。

02

给 body 加 boom 类

在 HTML 的 body 标签上加 class="boom"，这是主题样式生效的开关。没有这个类名，所有组件样式都不会起作用，就像电器没有插电一样。

03

用 b- 前缀的类名写组件

所有组件类都以 b- 开头：b-btn 是按钮、b-card 是卡片、b-grid 是网格。像搭积木一样组合 HTML 类名，不需要写任何自定义 CSS。

实际使用中最容易踩到的问题

五个必知的坑

01

Grid 必须两个类

b-grid-3 只设列数，不设 display:grid。必须写 class="b-grid b-grid-3"，漏掉 b-grid 的话网格不会生效，内容会垂直堆叠。

02

b-center 会覆盖方向

b-center 设的是 flex-row，如果和 b-stack 或 b-hero 混用，会把本该垂直排列的内容变成水平的。需要居中 + 垂直排列时用 b-stack 加 style。

03

data-reveal 别滥用

带 data-reveal 的元素默认透明度为零，只有滚动到视口内才显示。如果每个区块都加，页面初次加载时大片空白。只在 hero 区域少量使用。

04

data-count 是字面值

data-target="347" 加 data-suffix="%" 会显示 347%，不是 3.47%。想要小数就写 data-target="3.47"。

05

不要写自定义 CSS

boom 有 50 多个组件类和工具类，大部分需求都能组合解决。先查有没有现成的类，实在没有再用极少量 inline style。
