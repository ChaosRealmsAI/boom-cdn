常见陷阱

b-grid-3 → 没有 display:grid。必须搭配 b-grid b-grid-3
b-center → 是 flex-row，会覆盖 b-stack 的 column 方向
data-reveal → 默认 opacity:0，别滥用，只加 hero 区域
data-target="347" → 显示 347%，要小数写 3.47
